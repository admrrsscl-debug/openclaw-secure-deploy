# Changelog

Todos los cambios notables en este repositorio de instalación segura de OpenClaw.

## [1.0.0] - 2026-03-06

### Añadido
- Scripts de instalación para Windows, macOS y Ubuntu.
- Configuración endurecida `openclaw.json.hardened` con pairing, allowlists, sandbox y herramientas restringidas.
- Guías detalladas de instalación para cada sistema operativo.
- Documentación de soporte remoto seguro con Tailscale.
- README principal con instrucciones claras.
- Licencia MIT.

### Características de seguridad incluidas:
- Gateway solo local (loopback) por defecto.
- Autenticación por token.
- Políticas de pairing para nuevos contactos.
- Grupos con lista blanca (allowlist).
- Perfil de herramientas "messaging" restrictivo.
- Sandbox Docker para agentes.
- Configuración de firewall opcional.
- Auditorías de seguridad automáticas.
